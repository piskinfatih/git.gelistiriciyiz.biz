Alt `git` komutları için yardım almak

(Sub Commands) kullanımı

help

Alt komutlar ile ilgili yardım almak için; 

git help <KOMUT>
git --help <KOMUT>
git <KOMUT> --help

Örnek;

git help commit
git --help commit
git commit --help
