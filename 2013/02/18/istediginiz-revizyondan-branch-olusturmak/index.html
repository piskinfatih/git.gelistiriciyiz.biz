İstediğiniz revizyondan `branch` oluşturmak

branch

Herhangi bir revizyondan branch oluşturabilirsiniz. 

git branch BRANCH_ADI REVIZYON
git checkout -b BRANCH_ADI REVIZYON

İlk örnekte, açtığınız branch’e git checkout BRANCH_ADI ile geçiş yapabilirsiniz. İkinci örnek; branch’i oluşturur ve hemen oluşan yeni branch’e checkout yapar. İkinci örnekteki kullanım daha pratik bir kullanım şeklidir.

Örneğin;

git checkout -b testing_branch 8bab7c4

gibi bir şekilde istediğiniz bir revizyondan branch oluşturabilirsiniz.
