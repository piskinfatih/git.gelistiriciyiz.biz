Kısa ve öz git-status;
READ_MORE

```bash
git config --global alias.sts "status -sb"
```
Eğer işlemi elle yapmak isterseniz, `~/.gitconfig` dosyasına;

    [alias]
    sts = status -sb

ekleyebilirsiniz. Daha sonra da; `git sts` şeklinde kullanabilirsiniz.
